# TIFT engine

This is the core component of TIFT

## Building

```
$ tsc
```

## Running tests
```
$ jest
```

### Running all tests in a particular file
```sh
$ jest out/test/enginebuildertest.js
```

### Running a single test case
```sh
$ jest -t 'name of test case to run'
```

## Linting
```sh
$ npm run lint
```
